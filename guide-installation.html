Guide d'installation

Installer Woldius Solar Application sur votre téléphone

Woldius Solar Application fonctionne directement dans votre navigateur — aucun store, aucun téléchargement. En l'ajoutant à votre écran d'accueil, vous obtenez une icône comme une vraie application, en un geste.

Android (Chrome)

  1. Ouvrez woldius-solar-application.html dans Chrome sur votre téléphone.
  2. Appuyez sur le menu ⋮ (trois points) en haut à droite.
  3. Sélectionnez « Ajouter à l'écran d'accueil » (ou « Installer l'application » si Chrome le propose).
  4. Confirmez — l'icône Woldius apparaît sur votre écran d'accueil, prête à l'emploi.

iPhone / iPad (Safari)

  1. Ouvrez le lien dans Safari (obligatoire — Chrome ou autre ne propose pas cette fonction sur iOS).
  2. Appuyez sur le bouton Partager (carré avec une flèche vers le haut), en bas de l'écran.
  3. Faites défiler les options et choisissez « Sur l'écran d'accueil ».
  4. Appuyez sur « Ajouter » en haut à droite. L'icône Woldius apparaît sur votre écran d'accueil.
Sur iPhone, l'icône ne peut être ajoutée que depuis Safari — c'est une limitation d'Apple, pas de Woldius Solar Application.

Avant votre première utilisation

L'accès est soumis à autorisation : demandez un accès sur ce site, utilisez l'essai gratuit de 14 jours, ou entrez un code d'accès reçu par e-mail — directement depuis l'écran d'accueil du logiciel.
